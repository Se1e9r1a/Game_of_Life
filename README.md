# Game_of_Life
«Жизнь» (англ. Game of Life) — клеточный автомат, придуманный английским математиком Джоном Конвеем в 1970 году. Это игра без игроков, в которой человек создаёт начальное состояние, а потом лишь наблюдает за её развитием.

Механики:
1) Отрисовка сетки
2) Отдаление приближение
3) Перетаскивание "камеры" на ПКМ
4) Установка/Убирание клеток на ЛКМ с возможностью зажатия
5) Горячие клавиши:
    1) С/С - очистить поле от клеток
   2) N/Т - Перейти на следующее поколение
   3) SPACE - Остановить/ Продолжить процесс
   4) +/- для управления темпом симуляции.
6) Отключение отрисовки сетки при сильном отдалении
7) Оптимизация: Сетка отрисовывается только в видимом радиусе, расчёт происходит только живых клеток, а не всей сетки
8) Бесконечно поле
9) Оптимизация: Отрисовка происходит только живых клеток в пределах видимого окна
10) Сделал UI элементы (Пазуа, следующая генерация, увеличение/уменьшение скорости генерации)
11) Возврат фокуса на основное приложение (чтобы не ломались кнопки при нажатии)
12) Подсветка кнопки space (Когда генерация остановлена)
13) Кастомный движок размещения (Placement Engine): Полный отказ от системного Drag-and-Drop в пользу собственной логики отрисовки, что убрало задержки интерфейса.
14) Механика «Призрачного превью»: При выборе паттерна он следует за курсором в полупрозрачном виде, показывая, где именно появятся клетки.
15) Библиотека паттернов: Боковая панель со списком готовых классических фигур (глайдеры, блоки и т.д.).
16) Вращение фигур: Возможность повернуть выбранный паттерн на 90 градусов нажатием клавиши R прямо во время размещения.
17) Отмена установки: Выход из режима размещения фигуры нажатием ПКМ без изменения состояния поля.
18) Контекстное колесо мыши: Автоматическое переключение роли колесика — зум в обычном режиме и вращение фигуры, если она «в руках».
19) Пауза при установке: Автоматическая остановка симуляции при выборе паттерна для точного позиционирования и возобновление после клика.


## Инструкция по запуску
# Создаем виртуальное окружение
python -m venv venv
# Активируем его (Windows)
venv\Scripts\activate
# Установка необходимых библиотек (PyQt6)
pip install -r requirements.txt
# Запуск приложения
python main.py

Действие	Управление
Рисование	Зажать ЛКМ и вести по полю
Навигация (Камера)	Зажать ПКМ и двигать мышь
Масштаб (Zoom)	Колесо мыши
Пауза / Старт	Space (Пробел)
Шаг вперед	N
Очистка поля	C
Поворот фигуры	Колёсико мыши (в режиме размещения)
Скорость симуляции	+ и -
