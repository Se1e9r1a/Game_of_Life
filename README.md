# Game of Life (Игра «Жизнь»)

«Жизнь» (англ. Game of Life) — клеточный автомат, придуманный английским математиком Джоном Конвеем в 1970 году. Это игра без игроков, в которой человек создаёт начальное состояние, а потом лишь наблюдает за её развитием.

---

## 🎮 Управление

| Действие | Управление |
| :--- | :--- |
| **Рисование** | Зажать **ЛКМ** и вести по полю |
| **Навигация (Камера)** | Зажать **ПКМ** и двигать мышь |
| **Масштаб (Zoom)** | **Колесо мыши** |
| **Пауза / Старт** | `Space` (Пробел) |
| **Шаг вперед** | `N` |
| **Очистка поля** | `C` |
| **Поворот фигуры** | **Колесо мыши** (в режиме размещения) |
| **Скорость симуляции** | Клавиши `+` и `-` |

---

## ✨ Ключевые механики

1.  **Отрисовка сетки:** Визуальное отображение игрового поля.
2.  **Навигация:** Свободное отдаление, приближение и перетаскивание «камеры» на ПКМ.
3.  **Интерактивность:** Установка и убирание клеток на ЛКМ с возможностью зажатия для рисования.
4.  **Горячие клавиши:**
    * `C` / `С` — очистить поле.
    * `N` / `Т` — перейти на следующее поколение.
    * `SPACE` — остановить или продолжить процесс.
    * `+/-` — управление темпом симуляции.
5.  **Адаптивная графика:** Отключение отрисовки сетки при сильном отдалении для экономии ресурсов.
6.  **Бесконечное поле:** Отсутствие жестких границ игрового пространства.
7.  **Оптимизация обсчета:** Сетка отрисовывается только в видимом радиусе; расчет происходит только для живых клеток.
8.  **Оптимизация рендеринга:** Отрисовка только живых клеток в пределах видимого окна.
9.  **Интерфейс (UI):** Панели управления паузой, шагами и скоростью генерации.
10. **Стабильность управления:** Автоматический возврат фокуса на основное приложение (предотвращает залипание кнопок).
11. **Индикация:** Визуальная подсветка кнопки Space, когда генерация остановлена.
12. **Custom Placement Engine:** Кастомный движок размещения паттернов без использования системного Drag-and-Drop (убирает задержки интерфейса).
13. **Призрачное превью:** Полупрозрачное отображение паттерна под курсором перед установкой.
14. **Библиотека паттернов:** Боковая панель со списком классических фигур (глайдеры, блоки и т.д.).
15. **Вращение фигур:** Поворот паттерна на 90 градусов колесиком мыши во время размещения.
16. **Отмена установки:** Выход из режима размещения фигуры нажатием ПКМ.
17. **Контекстное колесо мыши:** Автоматическое переключение: зум в обычном режиме и вращение фигуры, если она выбрана.
18. **Пауза при установке:** Автоматическая остановка симуляции при выборе паттерна для точного позиционирования.
19. **Система тем:** Динамическое переключение визуальных стилей (Classic Green, Cyberpunk Red, Polar White).
20. **Музыкальное сопровождение:** Фоновый эмбиент, зацикленный для создания атмосферы погружения.
21. **Система помощи:** Кнопка `?` в меню, открывающая описание управления без перегрузки интерфейса.
22. **Включение/отключение музыки:** Кнопка 🔊 в нижнем правом углу управляет звуковым сопровождением.
23. **Очистка статистики:** При возвращении в главное меню все данные о симулыции сбрасываются.
24. **Возвращение зума:** При возвращении в главное меню камера возращается в начальное расположение с нужным значением Zoom.

## 🚀 Инструкция по запуску

### Вариант 1: Через исполняемый файл (Windows)
1. Перейдите в папку `dist`.
2. Запустите **`GameOfLife.exe`**.
*Все ресурсы (иконки, музыка) вшиты в файл и не требуют установки Python.*

### Вариант 2: Исходный код (Python)
Для работы программы необходимо наличие Python 3.10+ и установленных зависимостей.

1. **Создание виртуального окружения:**
   ```bash
   python -m venv venv
2. **Активация окружения (Windows):**
   ```bash
   venv\Scripts\activate
3. **Установка необходимых библиотек (PyQt6):**
   ```bash
   pip install -r requirements.txt
4. **Запуск приложения:**
   ```bash
   python main.py
