# Game_of_Life
«Жизнь» (англ. Game of Life) — клеточный автомат, придуманный английским математиком Джоном Конвеем в 1970 году. Это игра без игроков, в которой человек создаёт начальное состояние, а потом лишь наблюдает за её развитием.

Механики:
1) Отрисовка сетки
2) Отдаление приближение
3) Перетаскивание "камеры" на ПКМ
4) Установка/Убирание клеток на ЛКМ с возможностью зажатия
5) Горячие клавиши:
    1) С/С - очистить поле от клеток
   2) N/Т - Перейти на следующее поколение
   3) SPACE - Остановить/ Продолжить процесс
6) Отключение отрисовки сетки при сильном отдалении
7) Оптимизация: Сетка отрисовывается только в видимом радиусе, расчёт происходит только живых клеток, а не всей сетки
8) Бесконечно поле
